    getGJSongInfo.php

    Gets info about a newgrounds song.

    Parameters

    Required Parameters

    songID - The newgrounds ID of the song you want information about

    secret - Wmfd2893gb7

    Optional Parameters

    N/A

    Response

    A song object with ~|~ as the delimiter

    Example

    Python

    import requests

data = {
    "secret": "Wmfd2893gb7",
    "songID": 803223
}

req = requests.post("http://boomlings.com/database/getGJSongInfo.php", data=data)
print(req.text)
